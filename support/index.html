Common fixes

iCloud sync

  • Make sure iCloud Drive is enabled on the device.
  • Confirm the same Apple ID is used on devices you expect to sync.
  • Open DadYolked once after enabling sync so iOS can refresh app storage.

Health, Dad Recovery, and Dad Readiness

  • Health access is optional and private.
  • Grant sleep and step permission in iOS Settings → Health if you want recovery insights.
  • Pregnancy profiles use Dad Readiness; post-birth profiles use Dad Recovery.

Siri, widgets, and Apple Watch

  • If quick actions look stale, open DadYolked once so shared app data refreshes.
  • Check notification and Live Activity permissions in iOS Settings.
  • Use the Apple Watch app for fast baby logs without opening the phone.

Universal Search

  • Try searches like “sleep location”, “medicine”, “kick”, “appointment”, or “milestone”.
  • Search results depend on what you have logged locally in the app.
  • If a result is missing, check the child profile and date range you are viewing.
